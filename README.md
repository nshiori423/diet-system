# Diet管理アプリ（PHP）

職業訓練（ポリテクセンター福岡 ICTエンジニア科）の課題として作成した、
PHPとMySQLを使用したダイエット管理Webアプリです。

## アプリ概要
ユーザーが体重や目標を管理し、日々の変化をグラフで確認できるアプリです。

## 主な機能
- 新規ユーザー登録
- ログイン機能
- プロフィール編集
- 体重の記録
- グラフ表示

## 使用技術
- PHP
- MySQL
- HTML / CSS
- JavaScript（グラフ表示）

## 工夫した点
- パスワードをハッシュ化して保存するように実装しました
- 入力チェックやエラーメッセージを表示するようにしました
